使用说明

1、将googlemapsV3目录复制到FCK的插件目录

2、修改FCK配置文件

注册googlemapsV3

FCKConfig.Plugins.Add( 'googlemapsV3', 'zh-ch') ;

googlemapsV3添加至工具栏

FCKConfig.ToolbarSets["Basic"] = [
['Bold','Italic','-','OrderedList','UnorderedList','-','Link','Unlink', '-','googlemapsV3','-','About']
] ;

配置插件默认值（可选）

  • FCKConfig.GoogleMaps_Width : 默认地图宽度
  • FCKConfig.GoogleMaps_Height : 默认地图高度
  • FCKConfig.GoogleMaps_CenterLat : 地图中心点经纬度之经度
  • FCKConfig.GoogleMaps_CenterLon :地图中心点经纬度之纬度
  • FCKConfig.GoogleMaps_Zoom : 缩放级别
  • FCKConfig.GoogleMaps_MarkerPoints : 默认标签
示例 FCKConfig.GoogleMaps_Width = "600";
FCKConfig.GoogleMaps_Height = "380";
FCKConfig.GoogleMaps_CenterLat = "31.22473";
FCKConfig.GoogleMaps_CenterLon = "121.52956";
FCKConfig.GoogleMaps_Zoom = "16";
FCKConfig.GoogleMaps_MarkerPoints = new Array({ lat: 31.22457, lon: 121.5292, title: '我的工作地点' }

3、修改googleMaps.html页面配置提示文字FCK编辑器

文件位于googlemapsV3/dialog下

修改下列语句 配置为自己的FCKeditor目录 <script type="text/javascript" src="/FCKeditor/fckeditor.js"></script> 修改下列语句 配置FCKeditor基本目录 FCKeditor.BasePath = "/FCKeditor/"; 修改下列语句 配置FCKeditor默认工具栏 oFCKeditor.ToolbarSet = "Basic";

4、清空缓存 开始使用
